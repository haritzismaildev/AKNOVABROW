FEATURES:
✓ Built-in Ad Blocker
✓ VPN Integration with multiple countries
✓ Bookmark Management (Import/Export)
✓ Multi-Tab Browsing
✓ YouTube Shorts Blocker
✓ Security Protection (Anti-malware, Anti-phishing)
✓ Privacy Protection
✓ Fast & Lightweight

SYSTEM REQUIREMENTS:
- Windows 10 or later (64-bit)
- 4 GB RAM (8 GB recommended)
- 500 MB free disk space
- Internet connection

INSTALLATION:
1. Run the installer
2. Follow the setup wizard
3. Launch AKNOVA Browser from desktop or start menu

USAGE:
- Navigate: Type URL or search terms in address bar
- VPN: Select country from dropdown and click Connect
- Ad Blocker: Automatic (toggle in menu)
- Bookmarks: Click star icon to bookmark current page
- Multi-Tab: Click "+ New Tab" button
